# **App Name**: KingServ PWA

## Core Features:

- Real-time Inventory Tracking: Live updates on stock levels with color-coded alerts (green, yellow, red).
- Smart Reorder System: Automated alerts when stock hits reorder points with suggested quantities.
- Repair Job Workflow: Complete workflow from job intake to completion with status updates and notifications.
- Customer Profiles: Detailed customer profiles with repair history, purchase history, and outstanding balances.
- Sales & POS System: Quick sales interface with barcode scanning, discounts, and multiple payment methods.
- Financial Dashboard: Real-time financial dashboard with key metrics like cash in hand, sales, and profit.
- Automated Repair Description: Uses input on a problem with a device to create a tool-assisted LLM suggestion of a probable cause of the device problem and some remedies to be attempted.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) to convey trust and professionalism, aligning with the 'Bespoke, Polished Distinctive design'.
- Background color: Light gray (#F5F5F5) for a clean, modern look. Ensures good contrast with primary elements.
- Accent color: Vibrant yellow (#FFEB3B) to highlight key actions and alerts. Provides a pop of energy against the calm blues and grays.
- Body and headline font: 'Inter', sans-serif, to ensure clarity, legibility and a neutral modern look.
- Minimalist line icons for key actions and status indicators. Consistent style to reinforce a premium feel.
- Clean, well-spaced layout with clear hierarchy. Focus on easy navigation and quick access to important information.
- Subtle transitions and animations to enhance user experience without being distracting. Provides feedback on interactions.
